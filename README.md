# Wouter's Developer Portfolio

Welcome to my personal developer portfolio — a showcase of my work, skills, and projects as a Microsoft-focused Developer.

---

## 🧭 Overview

This portfolio site was built to highlight my journey as a system developer transitioning into the Microsoft ecosystem. It includes selected projects demonstrating my experience with both front-end and back-end technologies.

**Tech Stack**
- **Frontend:** HTML5, CSS3, JavaScript, Tailwind CSS  
- **Hosting:** GitHub Pages (for this site), Azure App Service (for future apps)

---

## 🧱 Features

- Clean, responsive design built with Tailwind CSS  
- Accessible and SEO-friendly structure  
- Interactive components using vanilla JavaScript  
- Direct access to CV and portfolio projects  
- Modular file structure for easy maintenance

---

## 🗂️ Project Structure

- assets/ # CV and downloadable resources
- img/ # Images, logos, and icons
- index.html # Main landing page-
- styles.css # Global styling and Tailwind customization
- script.js # Interactive behavior and dynamic elements
- README.md # Project documentation
